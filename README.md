# Arduino Bluetooth Controlled Car

This project controls a 4-wheel robot car using Bluetooth commands sent from a mobile device.

## Hardware Used
- Arduino UNO
- L293D Motor Shield
- HC-05 Bluetooth Module
- 4 DC Motors
- Robot Chassis
- Battery Pack

## Commands
F → Forward  
B → Backward  
L → Left  
R → Right  

## Library
AFMotor Library (Adafruit Motor Shield)
